Тестирование приложения Яндекс.Самокат
Данный проект содержит набор автоматизированных тестов для проверки функциональности учебного сервиса Яндекс.Самокат — приложения для аренды самокатов. Тесты охватывают раздел с вопросами о важном, позитивный сценарий заказа самоката и проверку навигации по логотипам.

---

## Используемые фикстуры (conftest.py)
#driver
Создает экземпляр браузера Firefox перед каждым тестом и закрывает его после завершения.
Открывает главную страницу приложения перед началом теста.

---

## Тестовые данные (data.py)
# Urls
SCOOTER_URL — главная страница сервиса.

# FAQ_DATA
Список с индексами вопросов и ожидаемыми текстами ответов.
Используется для параметризации тестов в test_faq.py.

# ORDER_DATA_1 / ORDER_DATA_2
Два набора данных для позитивного сценария заказа:

- name — имя
- last_name — фамилия
- address — адрес
- metro — станция метро
- phone — телефон
- date — дата доставки
- rental_period — срок аренды
- color — цвет самоката
- comment — комментарий для курьера

## Локаторы (pages/locators.py)
Все локаторы вынесены в отдельный файл и сгруппированы по классам:

# MainPageLocators
- ORDER_BUTTON_TOP — кнопка "Заказать" вверху страницы
- ORDER_BUTTON_BOTTOM — кнопка "Заказать" внизу страницы
- QUESTIONS — список вопросов в разделе FAQ
- ANSWERS — список ответов на вопросы
- SCOOTER_LOGO — логотип Самоката
- YANDEX_LOGO — логотип Яндекса

# OrderPageLocators
- NAME_INPUT — поле ввода имени
- LAST_NAME_INPUT — поле ввода фамилии
- ADDRESS_INPUT — поле ввода адреса
- METRO_STATION — поле ввода станции метро
- PHONE_INPUT — поле ввода телефона
- NEXT_BUTTON — кнопка "Далее"
- DATE_INPUT — поле выбора даты
- RENTAL_PERIOD — поле выбора срока аренды
- COLOR_BLACK / COLOR_GREY — чекбоксы цвета самоката
- ORDER_BUTTON — кнопка "Заказать" в форме
- CONFIRM_BUTTON — кнопка "Да" в окне подтверждения
- SUCCESS_MESSAGE — сообщение об успешном заказе

# Базовый класс (pages/base_page.py)
Содержит общие методы для взаимодействия с элементами страницы

## Тесты (tests/)
# test_faq.py
Проверяет, что при клике на каждый вопрос открывается правильный текст ответа.
Тест параметризован данными из FAQ_DATA.

# test_order.py
Проверяет позитивный сценарий заказа самоката:

- через кнопку вверху страницы

- через кнопку внизу страницы
Использует параметризацию по двум наборам тестовых данных.
После успешного заполнения форм проверяет появление сообщения «Заказ оформлен».

# test_navigation.py
Проверяет навигацию по логотипам:

- клик по логотипу Самоката → открывается главная страница

- клик по логотипу Яндекса → открывается Дзен в новой вкладке
